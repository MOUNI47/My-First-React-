Simple React Components
Objective

The objective of this project is to understand the basics of React functional components and JSX by building a simple React application.

Project Description

This is a basic React application created using Vite.
The application demonstrates how a webpage can be built using multiple reusable functional components.

The project contains three main components:

Header

Content

Footer

Each component is written using JSX and combined in the main App.jsx file.

Technologies Used

React

JavaScript (JSX)

CSS

Vite
